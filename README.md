## Customer Management Systems Website part 1

#installation

npm install <br/>
npm i express <br/>
npm i cors <br/>
npm i sqlite3 <br/>
npm i body-parser <br/>
npm i email-validator <br/>

#run

npm start

#api check 

go to postman create Request <br/>
GET <br/>
http://127.0.0.1:8080/user <br/>
POST <br/>
http://127.0.0.1:8080/user <br/>
